## 🧠 Sobre o Projeto

A **Papely Landing** é uma aplicação front-end construída para apresentar serviços de forma visualmente impactante, com navegação fluida e componentes reutilizáveis.

O projeto segue boas práticas de desenvolvimento, organização de código e arquitetura baseada em componentes.

* ⚛️ React

* 📦 JavaScript (ES6+)

## 🚀 Como Executar o Projeto

### 1. Clonar o repositório

```
git clone https://github.com/jhonatandev-vieira/papely-landing.git
```

### 2. Acessar a pasta do projeto

```
cd papely-landing/papely-landing
```

### 3. Instalar dependências

```
npm install
```

### 4. Rodar em ambiente de desenvolvimento

```
npm run dev
```

---

## 🌐 Build para Produção

```
npm run build
```

Os arquivos serão gerados na pasta `dist/`.
